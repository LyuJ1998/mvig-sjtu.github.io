修改默认gcc版本

Written by 徐文强 with ♥ on February 19, 2020 in tricks

网上有一些switch的方案，但配置起来比较麻烦，有可能不同步。 我们还是直接改路径，这样在编译的时候会稳定一点。 sudo apt install gcc-6 g++-6

sudo ln -s /usr/bin/gcc-6 /usr/local/bin/gcc

sudo ln -s /usr/bin/g++-6 /usr/local/bin/g++
