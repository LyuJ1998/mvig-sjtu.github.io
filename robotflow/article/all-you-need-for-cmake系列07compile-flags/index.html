All You Need for CMake系列07——Compile Flags

准备代码

代码结构如下：

.
├── CMakeLists.txt
├── main.cpp

main.cpp:

#include <iostream>

int main(int argc, char *argv[])
{
   std::cout << "Hello Compile Flags!" << std::endl;

   // only print if compile flag set
#ifdef EX2
  std::cout << "Hello Compile Flag EX2!" << std::endl;
#endif

#ifdef EX3
  std::cout << "Hello Compile Flag EX3!" << std::endl;
#endif

   return 0;
}

CMakeLists.txt下面的内容如下：

cmake_minimum_required(VERSION 3.5)

# Set a default C++ compile flag
set (CMAKE_CXX_FLAGS "${CMAKE_CXX_FLAGS} -DEX2" CACHE STRING "Set C++ Compiler Flags" FORCE)

# Set the project name
project (compile_flags)

# Add an executable
add_executable(cmake_examples_compile_flags main.cpp)

target_compile_definitions(cmake_examples_compile_flags 
    PRIVATE EX3
)

编译流程

mkdir build
cd build
cmake ..
make

设置Compile Flags同样有两种方式，一种是cmake .. -DCMAKE_CXX_FLAGS="-DEX3"

如果需要更换编译器，比如clang

cmake .. -DCMAKE_C_COMPILER=clang-3.6 -DCMAKE_CXX_COMPILER=clang++-3.6 版本号可以根据实际安装的版本调整，用不用c的编译器也根据实际情况调整。

如果想把后续的make替换成ninja

cmake .. -G Ninja

关于ninja的使用就是另一回事情了。这是一个效率比Make高很多的工具。

修改语言版本

检查是否支持某个版本的语言

在CMakeLists中添加

include(CheckCXXCompilerFlag)
CHECK_CXX_COMPILER_FLAG("-std=c++11" COMPILER_SUPPORTS_CXX11)
CHECK_CXX_COMPILER_FLAG("-std=c++0x" COMPILER_SUPPORTS_CXX0X)

if(COMPILER_SUPPORTS_CXX11)#
    set(CMAKE_CXX_FLAGS "${CMAKE_CXX_FLAGS} -std=c++11")
elseif(COMPILER_SUPPORTS_CXX0X)#
    set(CMAKE_CXX_FLAGS "${CMAKE_CXX_FLAGS} -std=c++0x")
else()
    message(STATUS "The compiler ${CMAKE_CXX_COMPILER} has no C++11 support. Please use a different C++ compiler.")
endif()

设置标准语言版本

将c++语言版本设为11

set(CMAKE_CXX_STANDARD 11)

只针对某个特定的语言feature

启用auto这个功能（自c++11）

target_compile_features(target_name PUBLIC cxx_auto_type)
