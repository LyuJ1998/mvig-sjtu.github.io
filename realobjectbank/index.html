    Real Object Bank #

    Real Object Bank是一个面向机器人感知（Perception）和操作（Manipulation）研究的大型数据集，该数据集支持包括但不限于：

    • Object 6DoF Estimation
    • Object Articulation Estimation
    • Point Matching
    • Imitation Learning
    • Instance segmentation, Object detection, Semantic segmentation, Depth estimation等任务

    目前，该数据集处于原型验证阶段，当正式public available时，将

    • 包含物体xxxx个，物体知识标注类型，包括但不限于：
      • 3D model/sparse 3D model
        • geometry
        • texture
        • material
      • structure keypoint
      • affordance
    • 包含场景xxxx个，场景知识标注类型，包括但不限于：
      • scene model
        • geometry
        • texture
        • materials
      • affordance
    • 把物体和场景组合在一起时，即成为支持各种应用任务的数据集，此时需要针对各种任务单独进行标注。

    Why ROB？ #

    我们发现，在做机器人相关的perception task的时候，和一般意义下的CV中的perception task有点区别。前者是object-centric的，而后者是category-centric的。这样的差异一个直接的影响就是，前者需要你能access object，而后者不需要，从互联网上随便爬数据就可以。

    因此，进入机器人领域后，我们急需一个大型的，具有一定前瞻性的数据集，来支撑Robot Perception相关的研究。这样也有助于大家减少重复造轮子和标注成本。

    比起数据集，ROB更是一个Robot Perception相关的知识共享平台，我们会在ROB上打通Object-Scene-Task之间的生态闭环。
