    Scene #

    场景是放置物体和操作者的地方，它本身由不可动的物体组成（墙，天花板，地板等），如果有一些物体设定为是不可动的（比如窗），那么它可以作为scene的一部分，如果设定为是可动的，那么请按照Object处理。

    和物体一样，它可以通过现实或者仿真的形式接触。

    场景的标注格式（updating~） #

    有什么建议可以跟我说。

    标注项 标注格式 备注
    类名/category 文本 可以结合wordnet形成taxonomy
    几何模型/3D model mesh/sparse point cloud 如果mesh拿不到，可以用sparse点集代替
    建模方式/modeling method 文本 cad，scan，slam
    扫描仪型号/scanner model 文本 适用于建模方式为“scan”，“slam”的物体
    纹理/texture 图片
    材质/material 图片
    尺寸/size 数值
    光源/light source object 指定场景中的某个物体作为光源
    区域功能/Affordance 热力图 格式暂定
