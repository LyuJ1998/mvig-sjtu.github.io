    扫描流程 #

    真实物体均来自线下的扫描，我们使用的扫描仪包括 EinScan Pro 2X 2020，和 EinScan SP。

    对于一些难以扫描的物体，比如黑色，透明，高反光等，我们有选择地撒显影粉，对此类物体，其纹理的准确性无法保证。
