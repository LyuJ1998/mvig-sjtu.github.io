    Perception Task Zoo #

    本数据集设计之初就打算支持的视觉任务包括：

    • 物体6D pose估计/Object 6DoF Estimation
    • 物体关节估计/Object Articulation Estimation
    • 物体点匹配/Point Matching

    当然，这些数据集也可以同时用于上游任务，包括但不限于，分类，RGB-(D) 目标检测，RGB-(D) 实例分割等。 同时，为了做这个数据集，我们还开发了一些上游任务，会在相关工作发表后放进来。
